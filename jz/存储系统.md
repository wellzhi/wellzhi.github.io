# 存储系统的分类

## 按存储层次

## 按存储介质

- 半导体存储器（主存、Cache）
- 磁表面存储器（磁盘、磁带）
- 光存储器（光碟）

## 按存取方式

- <u>相联存储器</u>（Associative Memory）既可以按内容访问的存储器（Content Addressed Memory，CAM）
- <u>随机存取存储器</u>（Random Access Memory，RAM）：读写任何一个存储单元所需时间都相同，与存储单元所在物理位置无关
- <u>顺序存取存储器</u>（Sequential Access Memory，SAM）：读写一个存储单元所需时间取决于存储单元所在物理位置
- <u>直接存取存储器</u>（Direct Access Memory，DAM）：既有随机存取特性，也有顺序存取特性。先直接选取信息所在区域，然后按顺序方式存取。

​		最后两种（DAM、SAM）为<u>串行访问存储器</u>：读写某个存储单元所需时间与存储单元的物理位置有关。

## 按信息的可更改性

- 读写存储器（Read/Write Memory）：既可读，也可写（如：磁盘、内存、cache）
- 只读存储器（Read Only Memory）：只能读，<u>不能写</u>（如：实体音乐专辑通常采用CD-ROM，实体电影采用蓝光光源，BIOS通常写在ROM中）：注意：<u>实际上很多ROM也可以多次读写，只是比较麻烦</u>

## 按信息的可保存性

- 易失性存储器（主存、cache）：断电后，存储信息消失
- 非易失性存储器（磁盘、光盘）：断电后，存储信息仍在

- 破坏性读出存储器（如DRAM芯片，读出数据后要进行重写）：信息读出后，原存储信息被破坏。
- 非破坏性读出存储器（如SRAM芯片、磁盘、光盘）：信息读出后，原存储信息不被破坏。

# 存储器的性能指标

- 存储容量：存储字数 × 字长 （如1M ×  8位），MDR位数反应存储字长
- 单位成本：每位价格 = 总成本 / 总容量。
- 存储速度：<u>数据传输率（主存带宽） = 数据的宽度 / 存储周期</u>。数据的宽度即是存储字长
- <u>存取时间与存取周期（存取时间 + 恢复时间）</u>
